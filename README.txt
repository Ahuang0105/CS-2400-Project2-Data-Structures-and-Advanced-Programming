The IDE used to build this programs is Eclipse, the zip file "AlanHuang_p2.zip" file
contain 1 folder call "CS2400 Project 2". Please place the whole folder under your 
"eclipse- workspace" folder. Next open your Eclipse software and selete "File" at the very
top left corner, next click "open Projects from file System". Under the "import source" 
please selet the path to "CS2400 Project 2" folder and clip "finish". Also, the source
code is locate in the folder call "bin" .


The file StackInterface.java is the interface for Task 2 and Task 3 and implemented by 
BookStack.java, LinkedBookStack.java and VectorStack.java. 

(1) The pdf file "Project 2 Task 1" is for Task 1.

(2) ArrayStackTest.java is the Client program to test BookStack.java class for Task 2. 

(4) LinkedStackTest.java is the clinet programe to test LinkedBookStack.java class for Task 2.

(5) VectorStackTest.java is the clinet programe to test VectorStack.java class for Task 3.
